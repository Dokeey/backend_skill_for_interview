### Call by value vs Call by reference

Call by value는 인자로 받은 값을 복사하여 처리하기 때문에 원래 값이 보존되어 안전하다. 하지만 그만큼 메모리 사용량이 늘어난다.

Call by refernce는 인자로 받은 값을 복사하지 않고 직접 참조를 하기에 빠르다. 하지만 원래 값이 변경될 수 있다.
